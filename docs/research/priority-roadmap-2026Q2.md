# boby-grow-weapp 增长优化优先级路线图（2026Q2）

- 主目标：提升留存与活跃
- 配套报告：`docs/research/growth-optimization-report-2026-03-27.md`

## 1. 版本分期

## P0（第1-4周）
1. 导航重构：父母端 tabBar 改为 `总览/任务/审核消息/孩子/我的`
2. 结构收敛：`setting` 与 `family-manage` 合并
3. 去冗余：下线独立 `child/submit-wish` 主入口（保留 home 内嵌）
4. 审核效率：审核中心支持待办聚合、批量操作、红点提醒
5. 奖励闭环：新增兑换历史与规则解释

## P1（第5-10周）
1. 成长周报（家庭周报 + 孩子反馈摘要）
2. 连续达标挑战（3/7/14天）
3. 家庭协作提醒（待审提醒、超时提醒）
4. 任务模板（按年龄段和场景快速创建）

## P2（第11周+）
1. AI 家长反馈建议
2. 家庭模式分层（基础/进阶）
3. 成长画像深化（行为标签、反馈可视化）

## 2. 工作拆解（按域）

| 域 | 负责人建议 | 关键输出 |
|---|---|---|
| 前端信息架构 | 前端 | 新导航、路由收敛、入口统一 |
| 审核与消息 | 前后端 | 审核聚合页、待办API、红点逻辑 |
| 奖励闭环 | 前后端 | 兑换历史、文案解释、状态同步 |
| 增长机制 | 产品+前后端 | 周报、挑战、提醒策略 |
| 数据与分析 | 数据/后端 | 埋点、指标看板、实验报表 |

## 3. 每周验收（P0）

- Week1：导航方案定稿 + IA 映射图
- Week2：路由改造 + 设置聚合完成
- Week3：审核聚合与红点完成
- Week4：奖励闭环与回归测试完成

## 4. Go/No-Go 条件

- D7 留存连续 2 周不下降
- 审核处理中位时长下降趋势成立
- 关键流程（建档/提交/审核/兑换）通过率 >= 99%
